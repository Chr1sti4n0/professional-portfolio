Stox-ster

Deployed Project

This project allows the user to create a list of their top stocks. The user can search for a company stock ticker, create a personalized list and track the latest stock information. The user is also presented with news cards related the selected stocks. For this project, I was responsible for the user interface which included styling, responsive web-design and adding a dark mode feature. A third-party API was used to retrieve the stock and news information.

Statz-Tracker

Deployed Project

Statz-Tracker allows users to create a personal account or login, and select a video-game from the provided list and begin recording their gaming statistics. Technologies used in this project include Javascript, MVC Paradigm, Server-Side API, user authentication and MySQL DB. My responsibility was to develop the models and API routes for video game categories.

Gamers Link Up

Deployed Project

Gamers Link Up allows users to create a personal account/login and view or create listings for used video games. The majority of my focus was on the back-end (server-side) of the application. I was in charge of creating models, queries and mutations for game and user information. On the client side, I supported with updating and debugging the game form.

Weather Dashboard

Deployed Project

The Weather Dashboard application allows the user to search for any U.S. city and find the current temperature, wind speed, humidity level, UV index and 5-day forecast. The weather data was pulled from a third-party API (Open Weather API) which was called through URL parameters.
